# nonebot-eval
## 使用方法
```
https://github.com/fjh1997/nonebot-eval.git
cd nonebot-eval
#启动守护进程
sudo python3 watchdog.py 
sudo docker compose up -d 
```
然后启动![lagrange-onebot](https://github.com/LagrangeDev/Lagrange.Core)即可
```
./Lagrange.OneBot
```


## 示例
发送
```
code await feedback(Message('[CQ:dice]'))
```
